# SOC Operations Runbook

## Purpose
Define standard operating procedures for daily SOC monitoring, triage, investigation, and escalation.

---

## Monitoring Scope
- On-prem Windows endpoints
- Azure identity and resources
- SIEM platforms (Splunk and Microsoft Sentinel)

---

## Daily Analyst Responsibilities
- Review SIEM alerts
- Perform initial triage
- Escalate confirmed threats
- Document all actions

---

## Alert Triage Standards
1. Validate alert context
2. Identify affected assets
3. Determine severity
4. Decide response path

---

## Incident Handling
- Group related alerts into incidents
- Follow incident-specific runbooks
- Escalate based on impact

---

## Documentation Requirements
- Maintain investigation timelines
- Record containment actions
- Capture lessons learned

---

## Notes
This runbook reflects standard SOC operational practices.
